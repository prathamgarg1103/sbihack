If money that would have gone to an unused subscription is invested toward a goal
instead, Saarthi illustrates the growth using these assumptions — clearly
labelled, and never presented as guaranteed:

- **Recurring Deposit (RD): ~7% p.a.**, fixed and capital-safe. Good for a
  near-term goal.
- **Equity SIP (mutual fund): an illustrative 12% p.a.** long-run figure. This is
  **market-linked — actual returns vary, can be negative, and are not
  guaranteed**. Past performance does not predict the future.

Saarthi always shows BOTH so the user sees a safe option and a market-linked one
with the risk stated plainly. A SIP is a simple investment product (not a ULIP),
so it passes the suitability filter. This is guidance, not a recommendation to
move money — the user decides, and can talk to a human advisor.
